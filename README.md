# Rock Paper Scissors 

A simple command-line Rock Paper Scissors game built with Python.

The player competes against the computer by choosing rock, paper, or scissors. The computer generates a random move, and the game keeps track of the score until one player reaches 3 points.

## Features

- Player vs Computer gameplay
- Random computer move generation
- Score tracking
- Best of 3 match system
- Input validation
- Replay option

## Concepts Practiced

- Functions
- Return values
- While loops
- Conditional statements
- Lists
- Python random module
- Input validation
